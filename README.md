**Desenvolvedor focado em automação, dados e sistemas internos para operação.**

Trabalho criando ferramentas que reduzem tarefas manuais, organizam bases, integram sistemas e transformam processos repetitivos em fluxos mais confiáveis. Minha área principal está entre Python, análise de dados, automação de processos e desenvolvimento de sistemas web para uso operacional.

## Sobre Mim

Atuo na interseção entre desenvolvimento, dados e produtividade operacional. Tenho experiência criando soluções práticas para problemas reais: tratamento de planilhas, padronização de bases, validação de documentos, automações com interface gráfica, dashboards locais, integração com APIs e sistemas full-stack.

Gosto de construir software que resolve o problema inteiro, não apenas uma etapa isolada: entrada de dados, validação, processamento, interface, logs, exportação, documentação e melhoria contínua.

## Projetos Publicados

### [Internal Data Platform Demo](https://github.com/Ar3sMyth/internal-data-platform-demo)

Sistema full-stack para importação, padronização, consulta, exportação e auditoria de registros tabulares. Demonstra arquitetura com backend Django/DRF, frontend React, autenticação, logs, dashboards e fluxo de dados.

**Stack:** Django, Django REST Framework, React, Vite, Tailwind CSS, Recharts, SQLite.

### [CPF Data Quality Toolkit](https://github.com/Ar3sMyth/cpf-data-quality-toolkit)

Ferramenta desktop em Python para validação, normalização, comparação e exportação de bases tabulares contendo CPFs. Demonstra tratamento de dados, validação matemática, filtros dinâmicos e exportação em CSV/XLSX.

**Stack:** Python, Pandas, Tkinter, OpenPyXL.

### [CRM Audience Normalizer](https://github.com/Ar3sMyth/crm-audience-normalizer)

Conversor de bases de CRM para formatos de públicos personalizados. O projeto trabalha limpeza de telefones, e-mails, nomes, CEPs, agrupamento de contatos e relatórios de qualidade dos dados.

**Stack:** Python, Pandas, Tkinter, OpenPyXL.

### [Operations Dashboard Demo](https://github.com/Ar3sMyth/operations-dashboard-demo)

Dashboard web local para acompanhar registros operacionais em formato Kanban, com indicadores, filtros, edição, visão tabular, relatórios e importação/exportação JSON.

**Stack:** HTML, CSS, JavaScript, LocalStorage.

### [Controle de Estoque](https://github.com/Ar3sMyth/controleEstoque)

Projeto acadêmico desenvolvido durante o curso de Análise e Desenvolvimento de Sistemas, mantido como registro da evolução em lógica de programação, CRUD e modelagem de regras simples de negócio.

**Stack:** Fundamentos de programação e lógica aplicada.
