# Andrey Soares

**Desenvolvedor focado em automação, dados e sistemas internos para operação.**

Construo ferramentas que reduzem trabalho manual, organizam bases, automatizam fluxos repetitivos e transformam processos operacionais em sistemas mais confiáveis. Minha atuação combina Python, análise de dados, automação, interfaces internas e desenvolvimento web full-stack.

---

## Perfil Técnico

- Automação de processos com Python
- Tratamento, validação e padronização de dados com Pandas
- Importação e exportação de CSV, XLSX e JSON
- Interfaces desktop com Tkinter, CustomTkinter e PySide6
- Automação web com Selenium e BeautifulSoup
- Backend com Django e Django REST Framework
- Frontend com React, Vite, Tailwind CSS e Recharts
- Dashboards operacionais, APIs, logs e rastreabilidade
